# Games Portal — games.lukeinglis.com

## Project
- Next.js 16 App Router with TypeScript and Tailwind CSS v4
- Deployed on Vercel at games.lukeinglis.me
- Retro game portal aesthetic inspired by classic sites like AddictingGames and Miniclip
- Aggregates mini-games from across Luke's sports sites

## Concept
A nostalgic games portal that serves as a hub for all mini-games built across various sports sites:
- **F1 Fantasy** — f1.lukeinglis.com
- **World Cup 2026 Fantasy** — worldcup.lukeinglis.me
- **Football Fantasy** — football.lukeinglis.com

The design should evoke early 2000s flash game portals: bold colors, game thumbnails in a grid, categories, featured games, playful typography. But built with modern tech and responsive design.

## Design Direction
- Old school game portal vibes: think Miniclip circa 2005, AddictingGames, Newgrounds
- Bold, saturated colors with gradients
- Game cards with thumbnails, titles, play counts, ratings
- Categories sidebar or top nav
- Featured/popular games section
- Pixel-art or retro gaming inspired accents
- Dark background with bright, punchy accent colors
- Fun, not corporate
