# Games Portal — games.lukeinglis.me

## What This Project Is
A retro-themed games portal that aggregates mini-games from Luke's sports websites. Inspired by classic flash game portals like AddictingGames and Miniclip (circa 2005). Built with Next.js 16, TypeScript, and Tailwind CSS v4.

## Current State
Greenfield project — needs full scaffolding.

## Tech Stack
- Next.js 16 App Router
- TypeScript
- Tailwind CSS v4
- Deployed on Vercel

## Game Sources
- F1 Fantasy — f1.lukeinglis.com
- World Cup 2026 Fantasy — worldcup.lukeinglis.me
- Football Fantasy — football.lukeinglis.com

## Design Requirements
- Dark background with bold, saturated accent colors
- Game cards in a grid layout with thumbnails, titles, play counts, ratings
- Category navigation (Sports, Fantasy, Predictions)
- Featured/popular games section at the top
- Retro pixel-art or gaming-inspired accents and typography
- Mobile-first responsive design
- Nostalgic but polished — fun, not corporate

## Build Commands
- `npm run dev` — development server
- `npm run build` — production build
- `npm run lint` — ESLint
